# Тестовое задание Junior Laravel Developer
## Задача
Необходимо разработать небольшой CRUD-проект на Laravel для управления задачами.
Что нужно сделать
Создать простое приложение “Список задач”.
Функционал:
- Вывод списка задач
- Создание новой задачи
- Редактирование задачи
- Удаление задачи
- Просмотр одной задачи отдельно
- Изменение статуса задачи:
  - Новая
  - В работе
  - Выполнена

## Структура задачи
У каждой задачи должны быть поля:
- id
- title — название задачи
- description — описание
- status — статус
- created_at
 - updated_at

## Требования к выполнению
### Backend
Использовать Laravel
Реализовать CRUD
Использовать валидацию при создании и редактировании
### Frontend
Можно сделать в самом простом виде:
Blade-шаблоны
Bootstrap по желанию, но не обязателен
Важно, чтобы интерфейс был рабочий и понятный.
## Что должно быть в проекте
- Главная страница со списком задач
- Кнопка добавления новой задачи
- Форма создания задачи
- Форма редактирования задачи
- Кнопка удаления
- Страница просмотра одной задачи
### Дополнительно будет плюсом
Это не обязательно, но будет преимуществом:
- фильтрация задач по статусу
- поиск по названию
- pagination
